# Railfence Cipher

Railfence cipher is a cryptographic technique which is a form of transposition cipher.

Given a plain-text message and a numeric key, cipher/de-cipher the given text using Rail Fence algorithm. 

## Example:

Encryption
Input :  "hello world"
Key = 3
Output : horel ollwd
Decryption
Input : horel ollwd
Key = 3
Output : hello world
